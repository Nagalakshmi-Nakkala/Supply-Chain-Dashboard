# Supply Chain Analysis Dashboard

## Project Overview
This project analyzes supply chain performance 
using Python, SQL and Power BI to identify 
delivery trends, sales patterns, and order 
status insights.

## Tools Used
- **Python** — Data Cleaning & Preprocessing
- **SQL** — Data Querying & Analysis
- **Power BI** — Interactive Dashboard & Visualization

## Dataset
- Total Orders: 181K
- Total Sales: $36.78M
- Avg Delivery Days: 3.50
- Late Delivery %: 57.30

## Key Insights
- Standard Class has highest orders (108K)
- Europe & LATAM are top markets
- Late delivery rate is 57.30%
- Sales consistent across 2015-2017

## Workflow
Raw Data → Python (Clean) → SQL (Query) 
→ Power BI (Visualize) → Insights
